## Guía de Categorías

| Categoría | Cuándo usar |
|-----------|-------------|
| **Added** | Nuevas funcionalidades |
| **Changed** | Cambios en funcionalidades existentes |
| **Deprecated** | Funcionalidades que serán removidas pronto |
| **Removed** | Funcionalidades eliminadas |
| **Fixed** | Correcciones de bugs |
| **Security** | Correcciones de vulnerabilidades |

## Formato de Entradas

```markdown
- {Descripción clara del cambio} - #{task_id}
```

**Ejemplos:**
- Agregado endpoint para exportar reportes a PDF - #1234
- Mejorado rendimiento de consulta de inventario (50% más rápido) - #1235
- Corregido error al guardar productos con caracteres especiales - #1236
